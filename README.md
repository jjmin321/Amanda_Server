## Motive
- 학생들의 심심함과 외로움을 달래주자
- 학생들이 더 이상 외모 순위를 두고 싸우지 않게 하자 

## Function
- 계정을 생성하고 자신의 프로필 사진을 등록 
- 사진과 내용을 배경으로 사용자들이 서로서로 프로필을 보고 평점을 부여 
- 평점 순위를 바탕으로 자신의 프로필 사진의 위치가 메인페이지에서 달라짐.
- 다른 사용자의 프로필에 댓글을 달 수 있음.
